## Animation Guidelines

### Preferred Animations
- Fade in/out (opacity)
- Scale (subtle, < 1.05x)
- Slide (smooth, < 100px)

### Avoided Animations
- Rotate
- Flip
- Bounce
- Complex transforms

### Duration
- Quick: 150ms (hover)
- Normal: 300ms (transitions)
- Slow: 500ms (page loads)

## Implementation Notes

### CSS-in-JS Approach
All styling is inline for:
- Component portability
- No CSS file dependencies
- Type-safe styles
- Dynamic theming capability

### Gradient Syntax
```jsx
background: 'linear-gradient(135deg, #start 0%, #end 100%)'
```
- 135deg = diagonal (top-left to bottom-right)
- 0% = start color position
- 100% = end color position

### Flexbox Usage
```jsx
display: 'flex'
flexDirection: 'column' | 'row'
justifyContent: 'space-between' | 'center'
alignItems: 'center' | 'flex-start'
gap: '12px'
```
