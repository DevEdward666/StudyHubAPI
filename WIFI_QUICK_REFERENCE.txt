🔌 API ENDPOINTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Base URL: http://localhost:5143/api

Generate Password:
  POST /wifi/request
  Body: {"validMinutes": 60, "passwordLength": 8}

Validate Password:
  GET /wifi/validate?password=YOUR_PASSWORD

Redeem Password:
  POST /wifi/redeem?password=YOUR_PASSWORD

Router Whitelist:
  POST /router/whitelist
  Body: {"macAddress": "00:11:22:33:44:55", "ttlSeconds": 3600}
