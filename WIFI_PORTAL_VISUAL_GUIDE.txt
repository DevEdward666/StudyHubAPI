║  WORKFLOW EXAMPLE                                                ║
╠══════════════════════════════════════════════════════════════════╣
║                                                                  ║
║  1. ADMIN ACTION                                                 ║
║     ┌────────────────────────────────────────────────────────┐  ║
║     │ Admin opens WiFi Portal                                │  ║
║     │ Selects: 60 minutes, 8 characters                      │  ║
║     │ Clicks "Get + Copy"                                    │  ║
║     │ Password generated: "Xy7BqP2m"                         │  ║
║     │ Password copied to clipboard                           │  ║
║     └────────────────────────────────────────────────────────┘  ║
║                                                                  ║
║  2. ADMIN SHARES                                                 ║
║     ┌────────────────────────────────────────────────────────┐  ║
║     │ Admin tells customer: "WiFi password is Xy7BqP2m"      │  ║
║     │ Or shows QR code / writes on paper                     │  ║
║     └────────────────────────────────────────────────────────┘  ║
║                                                                  ║
║  3. CUSTOMER CONNECTS                                            ║
║     ┌────────────────────────────────────────────────────────┐  ║
║     │ Customer enters password in WiFi settings              │  ║
║     │ System validates and redeems password                  │  ║
║     │ Customer gets internet access                          │  ║
║     │ Password becomes invalid (one-time use)                │  ║
║     └────────────────────────────────────────────────────────┘  ║
║                                                                  ║
║  4. EXPIRATION                                                   ║
║     ┌────────────────────────────────────────────────────────┐  ║
║     │ After 60 minutes: password auto-expires                │  ║
║     │ Password removed from database by cleanup service      │  ║
║     └────────────────────────────────────────────────────────┘  ║
║                                                                  ║
